More On Web Address

On internet, there are numerous web sites; it is only with the help of web address that we can find a particular website.

Text based addresing system on internet is called Domain Name System (DNS). In this system, each website has a unique name such as: oracle.com., cbse.nic.in, etc.

The last three letters of the web address is said as Top Level Domain (TLD) that provides information about the root of the domain for the type of organization to which the address belongs to. It is alphabetical and helps to identify an address in Domain Name System.

Some common Top Level Domains are:

Abbreviations Used To Denote
.com Commercial Firms
.edu Universities/Education Firms
.gov Government Bodies/Organisation
.mil Military Organisation
.net ISP'S/Network
.org Non-Government/Non-Profitable Organisation
.co National & Multinational Companies
.int International Organisation

In DNS, codes of country can also be included. The code comprises of two letters. Some common codes of certain countries are:

Code Denotes the Country Flags
.au Australia
.jp Japan
.fr France
.in India
.us United Statesof America
.nz New Zealand
