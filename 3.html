More On Website

A website is a collection of one or more web pages which displays information with the help of web browser. A website can be compared to a book which contains different chapters in the form of web pages.

The first page of a website is called as 'Home Page'. The pages on a website are linked to one another and a person can navigate to any page by clicking on the links called 'Hyperlinks'.

Websites are hosted on the web server on rental basis by the 'Internet Services Provide'. In order to access them every website is provided with a unique name.The website may be owned by individuals, agencies, companies, colleges, universities, government bodies, etc.
